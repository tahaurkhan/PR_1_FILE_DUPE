# HashClean: Smart File Duplicate Detector and Organizer

## 📌 Overview
HashClean is a robust desktop application designed to intelligently scan, detect, and organize duplicate files on your system. By utilizing advanced cryptographic hashing, it ensures 100% accuracy in identifying duplicate content, helping users reclaim valuable disk space safely and efficiently. 

## ✨ Key Features
* **Accurate Duplicate Detection:** Uses SHA-256 hashing logic to compare file contents, ensuring no false positives based solely on file names.
* **Fast and Lightweight:** Built with an optimized architecture that handles large directories smoothly.
* **Smart Organization:** Helps categorize and manage files efficiently.
* **Safe Deletion (Recovery Bin):** Includes a built-in recovery bin feature, ensuring that deleted duplicates can be restored if needed before permanent removal.
* **Modern User Interface:** A clean, intuitive, and responsive UI built for seamless user experience.

## 🛠️ Tech Stack
* **Language:** Java
* **User Interface:** JavaFX (designed with Scene Builder)
* **Database:** SQLite (for fast local metadata storage and file indexing)
* **Other Tools:** JDBC for database connectivity

## 🚀 Prerequisites
To run or build this project locally, you will need:
* Java Development Kit (JDK) 11 or higher
* JavaFX SDK
* SQLite JDBC Driver
* Maven or Gradle (if you are using a build tool)
